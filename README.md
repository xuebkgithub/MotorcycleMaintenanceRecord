# 🏍️ 摩托车维护记录应用

> **项目定位**：专业的摩托车维护保养记录管理工具

## 📊 核心业务功能

### 1. 车辆信息管理
- 记录摩托车品牌、型号、购买日期等基础信息

### 2. 维护保养记录
- 保养项目、时间、费用详细记录
- 维护历史统计和分析
- 保养提醒服务

## 🚀 项目目标

搭建一个**专业的、易用的**摩托车维护管理平台，帮助车主：

- 📝 **记录维护历史**：每次保养的详细记录
- 📈 **统计分析**：维护费用趋势、单次平均成本等

## 🛠️ 技术说明

### 目录架构
```
MotorcycleMaintenanceRecord/
├── app.js                  # 小程序入口逻辑
├── app.json                # 全局配置（页面路由、分包、组件等）
- 💰 **成本管理**：全面统计维护费用，分析使用成本

### 3. 数据安全保障
- 所有数据存储在用户本地设备
- 无需网络连接即可使用

## 🔧 开发规范

### 命名约定
- 页面目录：kebab-case（如 `maintenance-list`）
- **数据加密**：敏感信息本地加密存储
